Blue To-Do

Installation instructions
for Windows

Step 1 : Create a destination folder

Create a folder called "blue-to-do". The name is very important !

Step 2 : Download the application

Download "blue-to-do.exe" and save it in the folder we created above ("blue-to-do").

Step 3 : Copy the folder path

Copy the path to the folder "blue-to-do".

Step 4 : Open the environment variables panel

Open the configuration panel, and search "environment variable".
Click "Edit the environment variables for this account".

Step 5 : Edit the environment variable

Double-click on "Path".

Click on "New" and paste the folder path.

Step 6 : Confirm and test

Close all the windows, by clicking on "OK".

Open a terminal, by pressing "Windows + R", and typing "powershell".

Type "blue-to-do", and press "Enter".
